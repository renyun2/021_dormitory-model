用 Python 3.11 + Vue3 + PostgreSQL 构建一个企业职工宿舍入住管理系统，Docker Compose 部署。

技术栈：
- 后端：Python 3.11，FastAPI + SQLAlchemy + asyncpg
- 前端：Vue3 + Vite + Element Plus
- 数据库：PostgreSQL 16
- 部署：Docker Compose

核心功能：
1. 宿舍台账：录入宿舍楼（楼栋号/性别：男/女/混合/楼层/房间），录入房间（房间号/床位数/类型：4人间/6人间/单人间/当前状态：正常/维修中），床位逐一编号
2. 入住管理：员工申请入住（填写申请人/申请原因/期望入住日期），管理员分配具体房间+床位，录入入住记录（入住日期/押金金额/钥匙领取），房间实时显示床位占用状态
3. 调换房间：员工申请调换房间（原因/期望房间），管理员审批并指定目标床位，执行调换时更新入住记录，生成调换历史
4. 退住管理：员工申请退住（退住日期/退住原因），宿管检查房间（填写检查情况：有无损坏+损坏描述），退还押金（扣除损坏赔偿后的实退金额），记录钥匙归还
5. 费用管理：配置住宿费（按房间类型/月收费标准），按月批量生成应收账单，录入实收款项，欠缴提醒，年度住宿费汇总
6. 设施报修：宿舍住户提交报修（设施类型：水电/门窗/床铺/空调/日常用品/问题描述/紧急程度），维修人员接单处理，完成后确认，住户评价（1-5分）

数据表：dormitories、rooms、beds、checkin_records、fee_records
init.sql 预置10栋宿舍200个房间和住户入住记录。
docker compose up -d 后访问 http://localhost:8080。
